# MimoAgent 项目指令

## 语言规则
- 所有回复使用中文
- 所有生成的 md 文件名必须是中文

## 工作流规则

### 代码任务（Superpowers 模式）
只要需求跟写代码有关，必须调用子智能体（Agent）执行，包括但不限于：
- 新增功能、修改代码、修复 bug
- 重构、优化、迁移
- 写测试、修测试
- 配置变更、构建脚本修改

### 前端任务（UI Pro Max 模式）
所有前端相关内容，必须保证用户体验：
- 按钮点击必须有视觉反馈（loading、disabled、toast）
- 异步操作必须有进度指示
- 错误必须有用户可见的提示（不能静默吞掉）
- 破坏性操作必须有确认对话框
- 空状态必须有引导文案

### 计划阶段（反审流程）
生成计划后：
1. 开 4-6 个 Agent 反向重审计划
2. 根据重审结果优化计划
3. 关闭重审 Agent
4. 开 2-4 个 Agent 提出未来优化方向、遗漏点、缺陷
5. 总结建议加到计划末尾
6. 询问用户哪些可以接受

### 交付阶段（反审循环）
完成代码后：
1. 开 4-6 个 Agent 重审：聊天、计划、交付是否一致
2. 检查是否有偷工减料、不合理之处
3. 收集结果后关闭 Agent
4. while 交付结果存在明显问题:
   - 基于反审结果生成修改计划
   - 执行修改计划
   - 再次开启反审
5. 直到反审无明显问题后停止

## 项目结构
- `src/main/` - Electron 主进程
- `src/preload/` - 安全桥接层
- `src/renderer/` - React 前端
- `src/shared/` - 共享类型
- `engine/` - Agent 引擎（独立 npm 包）

## 常用命令
- `npm run build` - 完整构建
- `npm test` - 运行测试（43 个）
- `npm run lint` - 代码检查
- `npm run typecheck` - 类型检查
