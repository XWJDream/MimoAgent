## 安全设计

### 权限系统

- **三级权限模式**：suggest（建议）、auto-edit（自动编辑）、full-auto（全自动）
- **路径级规则**：敏感文件（.env、.git）自动拦截
- **工具预设**：plan 模式禁用写入和 Shell 执行

### 沙盒执行

- 支持 Docker 容器隔离执行 Shell 命令
- 可配置内存限制、CPU 限制、网络访问
- 默认关闭，需手动启用

### 数据安全

- API Key 仅存储在本地 `.env` 文件
- `.env` 文件被 `.gitignore` 排除，不会提交到仓库
- 应用启动时自动验证 API Key 有效性
- 不收集任何用户数据或遥测信息

### 工具安全

- Shell 工具内置 14 条危险命令拦截规则
- 文件操作前自动显示 Diff 预览
- Git 操作使用 `execFile` 替代 shell 字符串拼接，防止注入
